# Recommended Repositories for Testing Web3 Tools

## Smart Contract Development Tools

### Foundry Testing
Repository: https://github.com/foundry-rs/forge-std.git
Purpose: Standard library for Foundry projects
Why: Contains comprehensive test examples and is actively maintained

Repository: https://github.com/OpenZeppelin/openzeppelin-contracts.git
Purpose: Battle-tested smart contract library
Why: Industry standard with extensive security audits

## Security Analysis Tools

### Slither Testing
Repository: https://github.com/crytic/not-so-smart-contracts.git
Purpose: Examples of vulnerable Solidity contracts
Why: Specifically designed to demonstrate security vulnerabilities detectable by Slither

Repository: https://github.com/OpenZeppelin/openzeppelin-contracts.git
Purpose: Security analysis of production contracts
Why: Real-world contracts with known security properties

### Mythril Testing
Repository: https://github.com/ConsenSys/mythril-example.git
Purpose: Example contracts for Mythril analysis
Why: Created by Mythril developers for testing purposes

### Echidna Testing
Repository: https://github.com/crytic/echidna-examples.git
Purpose: Example contracts with Echidna properties
Why: Official examples from Echidna developers

## Infrastructure Tools

### Terraform Testing
Repository: https://github.com/terraform-aws-modules/terraform-aws-vpc.git
Purpose: AWS VPC infrastructure module
Why: Well-documented and widely used Terraform module

### Docker Testing
Repository: https://github.com/docker-library/hello-world.git
Purpose: Simple Docker example
Why: Minimal example to test Docker functionality

### Kubernetes Testing
Repository: https://github.com/kubernetes/examples.git
Purpose: Kubernetes example applications
Why: Official examples from Kubernetes project

## Monitoring & Policy Tools

### Prometheus Testing
Repository: https://github.com/prometheus/client_golang.git
Purpose: Go client library for Prometheus
Why: Contains example applications with metrics

### OPA Testing
Repository: https://github.com/open-policy-agent/library.git
Purpose: Collection of OPA policies and examples
Why: Official policy examples from OPA project

## Machine Learning Tools

### MLflow Testing
Repository: https://github.com/mlflow/mlflow-example.git
Purpose: Simple MLflow example project
Why: Official example from MLflow project
